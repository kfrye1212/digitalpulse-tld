# DigitalPulse TLD - NFT Upgrade Package

This package contains everything you need to upgrade your existing DigitalPulse TLD smart contract to support NFT functionality.

## What's Included

- **Updated Smart Contract** (`programs/digitalpulse_tld/src/lib.rs`) - With NFT minting
- **Cargo Configuration** (`programs/digitalpulse_tld/Cargo.toml`) - With Metaplex dependencies
- **Anchor Configuration** (`Anchor.toml`) - Build settings
- **Upgrade Scripts** - Automated deployment
- **Documentation** - Complete guides

## Quick Start

See `QUICK_START.txt` for the fastest way to upgrade.

## What This Adds

After upgrading, every domain registration will:
- ✅ Mint an NFT automatically
- ✅ Create Metaplex metadata
- ✅ Set 5% royalty
- ✅ Be tradable on NFT marketplaces
- ✅ Show in NFT wallets
